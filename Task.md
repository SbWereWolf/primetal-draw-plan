# Задание

## Первоначальная формулировка
Здание цеха состоит из нескольких этажей, каждый из которых 
некий многоугольник (не обязательно выпуклый, однако 
углы в нём 90 или 270) а так же возможно наличие «дырок».

Нужно написать / набросать код, который будет генерировать 
здание цеха (без стен). Конфигурация этажей задаётся в 
xml (структура на Ваше усмотрение).

## Уточнённая формулировка
На входе описание плоской сплошной "детали" и описание отверстий 
на этой детале.

Контуры всех отверстий состоят из прямолинейных отрезков.

Выдать чертёж детали без размеров.

 Сделать консольное приложение C# принимающее на вход xml с 
 описанием геометрии детали и отверстий. 
 На выходе должен быть графический файл с чертежом.
